How to install:

1. Install SOAP::Light in your perl installation (e.g. via cpan)
2. Simply put all HTML, Javascript, Perl and resource files in one single web directory.
3. Create a MYSQL database and execute the mysql.sql script in order to create appropriate structures.
4. Copy quotd-example.ini to qotd.ini and edit it contents according to your database settings.
5. Enter some quotes in your database (table qotd_quotes)
6. Check with qotd_client.pl script whether your server is working correctly.
7. Embed the HTML snippet in one of your web pages

HAPPY QUOTING!
